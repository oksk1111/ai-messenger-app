# ai_messenger

이 프로젝트는 Vite 기반의 React 메신저 앱입니다.

## 시작하기

1. 의존성 설치
```
npm install
```
2. 개발 서버 실행
```
npm run dev
```

## 주요 기능 (예정)
- 실시간 채팅
- 사용자 인증
- 친구 추가 및 목록 관리
- 채팅방 생성 및 관리

- Vite + React
